# cnl-mui-editor

[![Build](https://github.com/yuanzong/cnl-mui-editor/actions/workflows/build.yml/badge.svg)](https://github.com/yuanzong/cnl-mui-editor/actions/workflows/build.yml)

Rich text editor components built on top of Tiptap 2 and MUI.

## Development

- Install dependencies: `npm ci`
- Build the package: `npm run build`

## CI

Pushes trigger `.github/workflows/build.yml`, which installs dependencies with `npm ci` and runs `npm run build` on Ubuntu using Node.js 20.
